# @tadashi/svelte-loadable

It's a fork from https://github.com/kaisermann/svelte-loadable


## Install

```
$ npm i @tadashi/svelte-loadable
```


## Dev

To start the development:

```
bin/node/pkg.js
bin/node/zera -m pnpm
pnpm dev
```


## Test

To start the test:

```
npm test
```
